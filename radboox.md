### Project Overview

RadbooX is a platform designed for MikroTik users, offering a Radius Server application that implements Authentication, Authorization, and Accounting (AAA) systems for PPPoE and Hotspot networks. Key features of RadbooX include:

- **Radius as Network Access Management Protocol**: Utilizes Radius protocol to manage network access, providing high security and efficient control over network users.
- **Support for CHAP, MSCHAP, and PAP Encryption Types**: Supports various encryption types such as CHAP, MSCHAP, and PAP, ensuring user data security during transmission.
- **Enhanced Security and User Data Processing**: Ensures user data processing is secure and efficient, allowing network administrators to manage users more effectively.
- **Billing and Financial Management**: Apart from being a Radius Server, RadbooX can also be used for financial management or billing, offering a comprehensive solution for users to manage network finances efficiently.

With RadbooX, we provide adequate facilities and ease of service, becoming an integrated and comprehensive solution for your needs in managing networks and finances efficiently.

As part of this project, the focus was on developing a modern and intuitive user interface using React.js and seamlessly integrating it with APIs. The user interface prioritized responsiveness, user authentication, data visualization, and seamless API integration to enhance the overall user experience and facilitate seamless interaction with RadbooX's services.
